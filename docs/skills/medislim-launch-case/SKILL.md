# MediSlim Launch Case

Use this skill when the task is about:

- 产品上市预测案例
- GLP-1 赛道样板
- 客户展示材料
- 定价 / 渠道 / KOL / 直播 / 风险联动分析

## Workflow

1. Read [cases/medislim/case_data.json](../../cases/medislim/case_data.json).
2. Read [cases/medislim/results/full_analysis.json](../../cases/medislim/results/full_analysis.json).
3. Use the case to answer:
   - 目标人群是谁
   - 市场空间有多大
   - 定价如何权衡
   - 哪个渠道和内容最有效
   - 财务与风险是否可接受
4. Repackage into:
   - client-facing benchmark deck
   - report excerpt
   - go-to-market sample

## Guardrails

- 说明这是合成数据案例，不是现实销售结果。
- 优先引用案例中的结构化数字，而不是泛化结论。
