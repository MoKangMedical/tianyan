# Business Plan Positioning

Use this skill when the task is about:

- 商业计划书
- 中国市场定位
- 产品矩阵叙事
- 竞争分析
- 客户与定价结构

## Workflow

1. Read [docs/business_plan.md](../../docs/business_plan.md).
2. Pull the relevant sections:
   - Aaru 对比
   - 中国适配策略
   - 三眼产品矩阵
   - 目标客户与收入模型
   - 合规框架
3. Reframe for the audience:
   - 投资人：市场规模、差异化、商业模式
   - 客户：应用场景、价值、交付
   - 内部：产品优先级与边界

## Guardrails

- 强调商业模拟，不要把能力说成敏感政治预测。
- 中国适配是叙事核心，不是附录。
