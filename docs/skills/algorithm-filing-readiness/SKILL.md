# Algorithm Filing Readiness

Use this skill when the task is about:

- 算法备案准备
- 合规红线说明
- 数据安全与用户权益表述
- 向客户解释 Tianyan 的监管边界

## Workflow

1. Read [docs/algorithm_filing.md](../../docs/algorithm_filing.md).
2. Extract the parts relevant to:
   - 算法类型
   - 应用场景
   - 数据安全措施
   - 用户权益保障
   - 备案清单与时间表
3. Present the output as one of:
   - 备案底稿摘要
   - 客户合规说明
   - 内部准备清单

## Guardrails

- 明确说明 Tianyan 使用 100% 合成数据。
- 不要表述为政治预测系统。
- 不要扩展到未在材料中出现的监管承诺。
