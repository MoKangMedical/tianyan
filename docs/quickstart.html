Quick Start

5分钟内完成你的第一次市场预测

1

安装天眼

选择你喜欢的方式安装天眼。

pip install tianyan
docker pull mokangmedical/tianyan:latest
docker run -p 8000:8000 tianyan
git clone https://github.com/MoKangMedical/tianyan.git
cd tianyan
pip install -r requirements.txt
2

配置API Key（可选）

配置LLM API Key以获得更高质量的预测。不配置时自动使用规则引擎。

# .env 文件
TIANYAN_LLM_PROVIDER=deepseek
DEEPSEEK_API_KEY=sk-your-key-here
提示：不配置API Key也可以使用天眼，系统会自动降级为规则引擎模式。预测质量会略低，但仍然可用。
3

第一次预测

使用Python SDK进行你的第一次市场预测。

from tianyan import ConsumerEye, SyntheticPopulation

# 生成1000人的合成人群
pop = SyntheticPopulation(size=1000, region="全国")

# 创建预测引擎
eye = ConsumerEye()

# 预测产品上市反应
result = eye.predict_product_launch(
    product_name="元气森林·夏限定",
    price=5.5,
    category="饮料",
    selling_point="0糖0卡",
    channels=["天猫", "抖音"],
    target_population=pop
)

# 查看结果
print(f"购买意愿: {result.confidence:.1%}")
print(f"建议: {result.recommendations}")
4

使用CLI工具

或者使用命令行工具进行预测。

# 产品预测
python cli_v2.py predict \
  --product "元气森林" \
  --price 5.5 \
  --category 饮料 \
  --channels 天猫 抖音

# 对话式预测
python cli_v2.py conversation --interactive

# 启动API服务器
python cli_v2.py server --port 8000
5

调用REST API

通过HTTP API进行预测，适合集成到现有系统。

# 注册获取API Key
curl -X POST http://localhost:8000/api/v2/register \
  -H "Content-Type: application/json" \
  -d '{"name": "测试用户", "email": "test@example.com"}'

# 产品预测
curl -X POST http://localhost:8000/api/v2/predict \
  -H "Content-Type: application/json" \
  -H "X-API-Key: ty_your_key" \
  -d '{
    "product_name": "元气森林",
    "price": 5.5,
    "category": "饮料",
    "channels": ["天猫", "抖音"],
    "population_size": 1000
  }'
下一步：查看 API文档 了解所有可用端点，或查看 案例报告 了解完整输出格式。
