API Documentation

天眼 Tianyan v2.0 · 28 endpoints · RESTful JSON API

认证方式：所有 API 请求需要在 Header 中传递 X-API-Key。 免费用户每日 50 次，Pro 用户 500 次，Enterprise 用户 10,000 次。 通过 POST /api/v2/register 注册获取 API Key。

认证 Authentication

POST /api/v2/register

注册用户，获取 API Key。

参数类型说明
name requiredstring用户名
email requiredstring邮箱地址
rolestring角色: free / pro / enterprise
查看响应示例 
{
  "api_key": "ty_a1b2c3d4e5f6...",
  "name": "张三",
  "role": "free",
  "daily_limit": 50,
  "message": "请保存好你的API Key，它不会再次显示。"
}
GET /api/v2/health

健康检查，确认服务状态。无需认证。

查看响应示例 
{
  "status": "healthy",
  "service": "tianyan",
  "version": "2.0.0",
  "timestamp": 1714234567.89,
  "llm": {"provider": "deepseek", "model": "deepseek-chat"},
  "database": {"total_simulations": 42}
}

人群生成 Population

POST /api/v2/population

生成合成人群，用于后续预测分析。

参数类型说明
sizeinteger人群规模，默认 1000，最大 50000
regionstring地区: 全国 / 华东 / 华南 / ...
age_mininteger最小年龄，默认 18
age_maxinteger最大年龄，默认 65
查看响应示例 
{
  "success": true,
  "size": 1000,
  "summary": {
    "avg_age": 35.2,
    "gender_dist": {"男": 0.52, "女": 0.48},
    "avg_income": 15800,
    "top_archetypes": {"精打细算型": 220, "品质追求型": 180}
  },
  "sample": [
    {"age": 28, "gender": "女", "city_tier": "一线城市", "monthly_income": 18000}
  ]
}

产品预测 Prediction

POST /api/v2/predict

产品上市预测，返回购买意愿、渠道建议、KOL策略等。

参数类型说明
product_name requiredstring产品名称
price requiredfloat价格（元）
category requiredstring品类
selling_pointstring卖点
channelsarray渠道列表，默认 ["线上"]
population_sizeinteger人群规模，默认 1000
查看响应示例 
{
  "success": true,
  "product": "元气森林",
  "confidence": 0.72,
  "metrics": {
    "purchase_intent": 0.72,
    "price_acceptance": 0.68,
    "channel_preference": "小红书"
  },
  "segments": {
    "18-24岁女性": {"intent": 0.78, "pct": 0.25},
    "25-35岁白领": {"intent": 0.72, "pct": 0.30}
  },
  "recommendations": [
    "首月聚焦小红书种草",
    "建议定价5.5元，走性价比路线"
  ]
}
POST /api/v2/predict/async

异步预测，适用于大人群规模。返回任务 ID，通过 GET /api/v2/task/{id} 查询结果。

POST /api/v2/kol

KOL 效果预测，返回不同类型 KOL 的 ROI、转化率、建议。

POST /api/v2/compare

AB 产品对比，返回两个产品的置信度对比和推荐。

对话式预测 Conversation

POST /api/v2/conversation

多轮对话式预测，通过自然语言收集信息并逐步完善预测。

参数类型说明
message requiredstring用户消息
session_idstring会话 ID，首次调用可省略
查看响应示例 
{
  "session_id": "abc123",
  "stage": "product",
  "message": "好的，我来帮你分析「减重产品」的市场前景。",
  "suggestions": ["定价多少？", "目标人群是谁？"],
  "data": {"name": "减重产品", "category": "健康消费品"},
  "is_complete": false
}

数据查询 Analysis

GET /api/v2/predictions

查询历史预测记录。

参数类型说明
productstring按产品名筛选
limitinteger返回数量，默认 20
GET /api/v2/stats

平台统计信息，包括 LLM 状态、数据库统计、活跃会话数。

GET /api/v2/task/{task_id}

查询异步任务状态。

参数类型说明
task_id requiredstring任务 ID

快速开始 Quick Start

# 1. 注册获取 API Key
curl -X POST https://api.tianyan.dev/api/v2/register \
  -H "Content-Type: application/json" \
  -d '{"name": "你的名字", "email": "your@email.com"}'

# 2. 产品预测
curl -X POST https://api.tianyan.dev/api/v2/predict \
  -H "Content-Type: application/json" \
  -H "X-API-Key: ty_your_key" \
  -d '{"product_name": "元气森林", "price": 5.5, "category": "饮料"}'

# 3. 对话式预测
curl -X POST https://api.tianyan.dev/api/v2/conversation \
  -H "Content-Type: application/json" \
  -H "X-API-Key: ty_your_key" \
  -d '{"message": "我想推出一款减重产品"}'
