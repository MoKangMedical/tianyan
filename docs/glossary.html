Glossary

天眼核心术语表

A
Agent
智能体
天眼中的虚拟消费者。每个Agent拥有独立的人格、偏好和决策逻辑，能够模拟真实消费者的购买、评价、分享等行为。
C
ConsumerEye (消费眼)
消费者洞察引擎
天眼三眼产品矩阵之一，专注于产品上市预测、广告创意测试、定价策略优化和用户流失预测。
合成人群
Synthetic Population
基于国家统计局公开数据生成的虚拟消费者群体。100%合成数据，不涉及任何真实个人信息。
K
KOL预测
KOL Effect Prediction
预测不同类型KOL（头部主播、中腰部博主、素人种草号等）的带货效果，包括ROI、转化率和建议。
L
LLM适配器
LLM Adapter
统一的LLM接入层，支持DeepSeek、MIMO、OpenAI三种提供商，自动降级为规则引擎。
M
MarketEye (市场眼)
市场洞察引擎
天眼三眼产品矩阵之一，专注于行业趋势预测、竞品动态分析和投资风向判断。
麦肯锡式报告
McKinsey-style Report
天眼自动生成的专业报告格式，包含Executive Summary、风险矩阵、执行计划等完整结构。
P
PolicyEye (政策眼)
政策洞察引擎
天眼三眼产品矩阵之一，专注于政策影响评估、民意温度计和经济趋势研判（严格去政治化）。
S
场景推演
Scenario Simulation
将产品、定价、渠道、KOL、直播等变量放入同一套推演，模拟真实市场反应的过程。
T
TAM/SAM/SOM
Total/Service/Obtainable Market
市场分层分析框架。TAM是总潜在市场，SAM是可服务市场，SOM是可获得市场。
