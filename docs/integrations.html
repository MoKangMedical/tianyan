Integrations

将天眼集成到你的工作流中

🐍 Python SDK

直接在Python代码中使用天眼。

from tianyan import ConsumerEye, SyntheticPopulation

pop = SyntheticPopulation(size=1000, region="全国")
eye = ConsumerEye()
result = eye.predict_product_launch(
    product_name="产品名称",
    price=99.0,
    category="品类",
    selling_point="卖点",
    channels=["天猫", "抖音"],
    target_population=pop
)
print(f"购买意愿: {result.confidence:.1%}")
⚡ CLI 命令行

在终端中直接使用天眼。

# 产品预测
python cli_v2.py predict --product "元气森林" --price 5.5 --category 饮料

# 对话式预测
python cli_v2.py conversation --interactive

# 启动API服务器
python cli_v2.py server --port 8000
🌐 REST API

通过HTTP API集成到任何语言。

# Python
import requests
r = requests.post("http://localhost:8000/api/v2/predict",
    headers={"X-API-Key": "ty_your_key"},
    json={"product_name": "产品", "price": 99, "category": "品类"})

# JavaScript
fetch("http://localhost:8000/api/v2/predict", {
    method: "POST",
    headers: {"X-API-Key": "ty_your_key", "Content-Type": "application/json"},
    body: JSON.stringify({product_name: "产品", price: 99, category: "品类"})
})

# cURL
curl -X POST http://localhost:8000/api/v2/predict \
  -H "X-API-Key: ty_your_key" \
  -H "Content-Type: application/json" \
  -d '{"product_name": "产品", "price": 99, "category": "品类"}'
🐳 Docker

一键部署天眼服务。

# 构建并启动
docker-compose -f docker-compose.prod.yml up -d

# 或单独运行
docker run -d -p 8000:8000 \
  -e TIANYAN_LLM_PROVIDER=deepseek \
  -e DEEPSEEK_API_KEY=sk-your-key \
  mokangmedical/tianyan:latest
☁️ 云平台部署

支持主流云平台部署。

# AWS ECS
aws ecs create-service --cluster tianyan --task-definition tianyan:1

# 阿里云 ACK
kubectl apply -f k8s/deployment.yaml

# Render
render deploy --service tianyan
