# Investor Narrative

Use this skill when the task is about:

- 融资叙事
- 投资人一页纸
- 牵引数据表达
- 资金用途与里程碑

## Workflow

1. Read [docs/investor_deck.md](../../docs/investor_deck.md).
2. Extract:
   - 核心问题
   - 解决方案
   - 市场与竞争
   - 商业模式
   - 融资需求与里程碑
3. Rebuild into:
   - one-pager
   - investor memo
   - fundraise intro

## Guardrails

- 用投资语言表达，不用产品功能堆砌。
- 数据与牵引必须和现有材料一致。
