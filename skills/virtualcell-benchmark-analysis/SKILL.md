# VirtualCell Benchmark Analysis

Use this skill when the task is about:

- 单细胞模型 benchmark
- foundation model 横向比较
- 模型表现总结与对外叙事
- 将技术基准测试转成客户或投资人可读材料

## Workflow

1. Read [虚拟细胞/docs/benchmark_report.md](../../虚拟细胞/docs/benchmark_report.md).
2. Read [虚拟细胞/docs/benchmark_report.json](../../虚拟细胞/docs/benchmark_report.json) when precise task, model, or dataset numbers are needed.
3. Extract the answer in this order:
   - 基准测试覆盖了哪些任务与数据集
   - 哪些模型在什么任务上领先
   - 结果意味着什么，不意味着什么
   - 对技术路线、产品定位或合作沟通有什么启发
4. When needed, compare this snapshot with [虚拟细胞/docs/benchmark.md](../../虚拟细胞/docs/benchmark.md) to explain version changes.

## Guardrails

- 指标、任务和模型名称必须与原始 benchmark 一致。
- 明确区分 benchmark 表现与真实生产场景表现。
- 不要把实验结果扩展成未经验证的临床或商业承诺。
