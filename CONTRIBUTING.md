# 🤝 贡献指南

感谢你对本项目的关注！我们欢迎所有形式的贡献。

## 如何贡献

### 报告 Bug
使用 [Bug Report](https://github.com/MoKangMedical/tianyan/issues/new?template=bug_report.md) 模板提交问题。

### 功能建议
使用 [Feature Request](https://github.com/MoKangMedical/tianyan/issues/new?template=feature_request.md) 模板提出建议。

### 代码贡献

1. Fork 项目
2. 创建功能分支 (`git checkout -b feature/amazing-feature`)
3. 提交更改 (`git commit -m 'feat: add amazing feature'`)
4. 推送到分支 (`git push origin feature/amazing-feature`)
5. 创建 Pull Request

## 开发规范

### 提交信息格式
```
<type>(<scope>): <subject>

类型：
- feat: 新功能
- fix: 修复 Bug
- docs: 文档更新
- style: 代码格式（不影响功能）
- refactor: 重构
- test: 测试相关
- chore: 构建/工具相关
```

### 代码风格
- Python: PEP 8 / Black 格式化
- TypeScript: ESLint + Prettier
- 提交前运行 `lint` 检查

### 分支策略
- `main`: 稳定版本
- `develop`: 开发分支
- `feature/*`: 功能分支
- `fix/*`: 修复分支

## 社区准则

- 尊重所有参与者
- 建设性讨论
- 不接受任何形式的骚扰
