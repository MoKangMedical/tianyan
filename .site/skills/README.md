# Tianyan Skills Library

这些 skills 是把仓库内现有报告、案例和方法论沉淀成可复用能力的第一版。

原则：

- 一份核心材料，对应一个可触发的 skill
- skill 只保留如何使用这份材料的最短路径
- 细节留在原始报告或案例文件中
- 后续新增行业报告时，优先补一个新的 `skills/<name>/SKILL.md`，再更新 `content/skill-library.json`
