# Realistic Go-To-Market

Use this skill when the task is about:

- POC 路线
- 销售周期
- 现实落地策略
- 定价调整
- 从免费案例到付费客户

## Workflow

1. Read [docs/realistic_strategy.md](../../docs/realistic_strategy.md).
2. Focus on:
   - 五个错误假设
   - 三层数据策略
   - 免费 POC → 付费 → 平台化路径
   - 真实财务模型与定价逻辑
3. Output as:
   - GTM memo
   - stage plan
   - execution checklist

## Guardrails

- 以真实执行约束为先，不复述理想化增长故事。
- 先验证案例与信任，再谈规模化。
